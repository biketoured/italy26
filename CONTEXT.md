## static/js/main.js — SHARED JS

Loaded with `defer` on every page. Exports globals: currentLang.

### getCurrentPage() → string
Returns: 'index' | 'donate' | 'flaskpost' | 'post' | 'blog' | 'route' | 'sweden2024'
NOTE: contact page not yet added to getCurrentPage() — add if needed.
Detection order matters: 'post' checked before 'blog' (path includes both).

### getPrefix() → string
**CRITICAL — must always exist**
Returns '../' if URL includes '/blog/', else ''.
Used to build correct asset paths from blog/ subdirectory.
If this function is missing or broken, nav injection crashes silently on ALL pages.

### NAV_THEMES{} — nav colours per page
```
index/route/blog/post: rgba(30,61,47,0.96) green, stone links
sweden2024/donate:     rgba(74,122,150,0.97) blue, pale blue links
flaskpost:             rgba(42,31,18,0.96) dark brown, sand links
```
Edit here to change nav colours — never in HTML files.

### PAGE_IMAGES{} — scrollbar thumb per page
```
index→pizza.png, donate→heart.png, blog→wheel.png,
post→cheese.png, sweden2024→mayflower.png,
route→tomato.png, flaskpost→flaskpost.png
```

### injectNav(page)
Builds 3-slot grid nav:
- Left: .nav-home (hidden via visibility:hidden on index, not display:none)
- Centre: .lang-switcher with flag buttons (EN/SV/IT, always visible)
- Right: .nav-links ul (desktop) + .nav-hamburger button
- Also appends .nav-dropdown for mobile
- Hamburger toggle: closes on link click, outside click, and X click

### Nav link order
Donate → Blog → Route → Sweden 2024 → Links → Contact
Mobile menu: same minus Links. Current page link is hidden (li.hidden = display:none).
Each li has min-width:6rem so nav doesn't shift width between pages.

### injectFooter()
Fills #shared-footer: "Edvard Appelberg" logo + "Skellefteå, Sweden — Built with HTML & CSS"

### applyLanguage(lang)
- Swaps innerHTML of all [data-en] elements to data-{lang}
- Dispatches 'langchange' CustomEvent on document
- Saves to localStorage as 'preferred-lang'
- currentLang global is updated

### initLanguage()
- Event delegation on document for .lang-btn clicks
- Restores saved lang from localStorage on load

### injectPizzaScrollbar()
- Creates #pizza-scrollbar div, sets background = nav theme bg (opacity forced to 1)
- Spin: rAF loop, exponential decay after scroll stops
- Drag: pointer events + setPointerCapture
- Touch: touchstart on track for tap-to-jump

### Execution order (bottom of file)
```js
injectNav();
injectFooter();
initLanguage();
injectPizzaScrollbar();
```

### CRITICAL RULE — currentLang in page scripts
main.js loads with defer. Page scripts that need currentLang must:
```js
const lang = (typeof currentLang !== 'undefined' ? currentLang : null)
          || localStorage.getItem('preferred-lang') || 'en';
```
And re-run on language change via:
```js
document.addEventListener('langchange', myFunction);
```

## sync_posts.py — POST SYNC SCRIPT

Location: scripts/sync_posts.py
REPO_ROOT = two levels up from script = repo root
POSTS_DIR = REPO_ROOT/blog/posts
OUTPUT    = REPO_ROOT/blog/posts.json

### What it does
1. Scans all subdirs of blog/posts/ (skips MALL* and .* folders)
2. Skips folders with no en.md
3. Renames images alphabetically:
   - Files NOT matching ^(cover|photo\d+)\.(jpg|jpeg|png)$ are "raw"
   - Raw images sorted alphabetically → first=cover.jpg, rest=photo1.jpg, photo2.jpg...
   - Already-named files are untouched (manual naming wins)
4. Reads frontmatter, builds entry, sorts posts newest-first
5. Writes posts.json (never manually edit this file)

### posts.json entry structure
```json
{
  "slug": "dag1",
  "type": "journal",
  "date": "2026-05-10",
  "day": 1,
  "location": "Genova, Liguria",
  "km": 87,
  "elevation": 1240,
  "title": { "en": "...", "sv": "...", "it": "..." },
  "captions": { "photo1": "...", "photo2": "..." },
  "photos": 2
}
```

## CRITICAL ARCHITECTURE RULES

1. getPrefix() MUST exist in main.js. If missing/broken → nav crashes silently on all pages.

2. posts.json is AUTO-GENERATED. Never edit manually. Next sync overwrites it.

3. currentLang is set by deferred main.js. Page scripts must use the safe pattern:
   `(typeof currentLang !== 'undefined' ? currentLang : null) || localStorage.getItem('preferred-lang') || 'en'`

4. All scrollbar CSS in style.css only. Never inject scrollbar <style> from main.js (caused override bugs historically).

5. blog/ pages: getCurrentPage() returns 'blog' for both blog/index.html AND blog/post.html.
   The path check order matters: 'blog'+'post' → 'post', 'blog' alone → 'blog'.

6. donate.html: footer MUST be outside .page div (max-width:860px would constrain it).

7. meter-goal span in index.html has NO data-* attributes. Translation + number both set
   by JS meter script. Has langchange event listener to re-render on language switch.

8. story-full span in donate.html has NO fallback text between tags.
   Content only comes from data-* via applyLanguage(). Needs explicit color:var(--stone).

9. NAV_THEMES in main.js controls all nav colours. Never set nav colours in HTML files.

10. Scrollbar background is set dynamically in JS after nav injection:
    navTheme.bg with opacity forced to 1 (replaces last rgba value).

11. Hamburger breakpoint: 1024px. .nav-links hidden, .nav-hamburger shown.
    Dropdown inherits --nav-bg + --nav-border → automatically matches page theme.

12. contact.html profile photo: z-index:2 required to appear above banner ::after overlay (z-index:1).

13. Banner drift in contact.html: .banner-inner is 116% size (8% bleed).
    overflow:hidden on .contact-banner clips the drift. Never remove overflow:hidden.
